# CodeAlpha - Basic Network Sniffer

## 📌 Project Overview

This project is developed as part of the **CodeAlpha Cyber Security Internship**.

The objective of this task is to build a basic network packet sniffer using Python to capture and analyze network traffic packets. The program helps in understanding how data travels across networks and how different protocols work.

The sniffer captures live packets from the network and displays important information such as:

- Source IP Address
- Destination IP Address
- Protocol Type
- Payload Data

---

# 🚀 Features

✅ Capture live network packets  
✅ Display source and destination IP addresses  
✅ Detect network protocols (TCP, UDP, ICMP)  
✅ Display packet payload data  
✅ Real-time packet analysis  
✅ Understand packet structure and traffic flow  

---

# 🛠 Technologies Used

- Python 3
- Scapy Library
